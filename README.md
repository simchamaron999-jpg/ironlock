- ✅ **פשוט להעלות** - פשוט חבר GitHub
- ✅ **חינמי לגמרי** - 750 שעות/חודש
- ⚠️ השירות ינום אחרי 15 דקות

### 📖 קרא את זה:
👉 **[DEPLOY_RENDER.md](DEPLOY_RENDER.md)** - הוראות מלאות ל-Render
